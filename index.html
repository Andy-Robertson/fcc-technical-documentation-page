Introduction

The flexibox box layout was first introduced to CSS3 having been inspired by the flexibility of JavaScript frameworks. There are several key features associated with flexibox, these include:

  • Fit to any size screen;
  • Avoids unnecessary monopolization of space;
  • Less restrictive content flow compared to block and inline;
  • Items can be individually reordered to suit the layout;
  • Directional flow can be specified at the style level.
Display

Display specifies the type of rendering box used for an item.

display: flex;

Makes the flex items display inline.

display: flex-inline;

Makes the flex container display inline.

Flex Direction

Specifies how flex items are placed in the flex container.

flex-direction: row;

Text is arranged in a horizontal line from left to right.

flex-direction: row-reverse;

Text is arranged in a horizontal line from right to left.

flex-direction: column;

Text lines are stacked from top to bottom.

flex-direction: column-reverse;

Text lines are stacked from bottom to top.

Flex Wrap

Specifies whether flex items are forced onto a single line or wrapped onto multiple lines.

flex-wrap: nowrap;

Items stay on one line.

flex-wrap: wrap;

Items wrap onto other lines.

flex-wrap: wrap-reverse;

Items wrap in the opposite direction defined by flex-direction.

Justify Content

Defines how the browser distributes space between and around flex items along the main axis.

justify-content: flex-start;

Position items starting from the start.

justify-content: flex-end;

Position items starting from the end.

justify-content: center;

Position items around the centre.

justify-content: space-between;

Position items evenly, first item flush at the left and last item flush with the right.

justify-content: space-around;

Position items evenly with a half-sized space on each end.

justify-content: space-evenly;

Position items evenly with even space between them.

Align Items

Defines how the browser distributes space between and around flex items along the cross-axis.

align-items: stretch;

Stretch items to fit.

align-items: flex-start;

Align items starting from the start.

align-items: flex-end;

Align items starting from the end.

align-items: center;

Align items in the centre.

align-items: baseline;

Align items to baseline.

Align Content

Aligns a flex container’s lines within the flex container when there is extra space on the cross-axis.

align-content: flex-start;

Pack lines from the cross-axis start.

align-content: flex-end;

Pack lines to the cross-axis end.

align-content: center;

Pack lines around the cross-axis centre.

align-content: space-between;

Distribute lines along the cross-axis, the first item is flush with the start, the last is flush with the end.

align-content: space-around;

Distribute lines evenly along the cross-axis, all items have a half-size space on either end.

align-content: stretch;

Stretch lines to occupy the whole cross-axis.

align-content: space-evenly;

Distribute lines evenly along the cross-axis, Items all have equal space around them.

Order

Specifies the order used to lay out flex items in their flex container.

order: 0;

Accepts numerical values including negative numbers. Global values accepted including, inherit; initial; and unset; (initial value = 0)

Flex Grow

specifies the flex grow factor of a flex item. It specifies what amount of space inside the flex container the item should take up.

flex-grow: 0;

Numerical values accepted and global values including: inherit; initial; and unset; (initial value = 0).

Flex Shrink

Specifies the flex grow factor of a flex item. It specifies what amount of space inside the flex container the item should take up.

flex-shrink: 1;

Numerical values accepted along with glogal values including: inherit; initial; and unset; (initial value = 0).

Flex Basis

Specifies the flex basis which is the initial main size of a flex item. Determines the size o the content-box unless specified otherwise using box-sizing.

flex-basis: auto;

Widths can be specified in numerical values using em and px, (initial value = auto).

Flex

Shorthand property specifying the ability of a flex item to alter its dimensions to fill available space.

flex:​ 0 1 auto;

The above example is shorthand for the following, flex-grow: 0; flex-shrink: 1; flex-basis: auto;

Align Self

Defines how the browser distributes space between and around flex items along the cross-axis of their container.

align-self: flex-start;

Align to cross-start.

align-self: flex-end;

Align to cross-end.

align-self: center;

Center items in the cross-axis.

align-self: baseline;

Align items to baselines.

align-self: stretch;

Stretch items to fit.
